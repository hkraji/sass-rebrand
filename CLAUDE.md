# SaaS Rebrand - Vibe Kanban Demo

## Project Overview

A Next.js demo site rebranding saas.group to showcase Vibe Kanban's parallel agent capabilities.

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- vibe-kanban-web-companion (point-and-click component selection)

## Development

```bash
npm install
npm run dev
```

## Project Structure

```
src/app/
├── page.tsx        # Homepage
├── layout.tsx      # Root layout
└── globals.css     # Global styles
```

## Pages to Build (Vibe Kanban Demo)

The following pages can be built in parallel using Vibe Kanban:

1. **Homepage** - Hero, testimonials carousel, stats, acquisition criteria, contact form
2. **About Page** - Goals cards, core principles, FAQ accordion, team grid
3. **Family/Portfolio Page** - Featured companies, company grid, testimonials

## Design System

Based on saas.group:
- **Primary Color**: Orange (#E15C34)
- **Accent Color**: Purple (#A375FF)
- **Typography**: Inter (headings), Roboto (body)
- **Patterns**: Full-width sections, card layouts, gradient overlays
